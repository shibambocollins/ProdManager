# Items Manager - Backend

Backend for Product-Management/Items-Management.
It is a simple project for learning purposes and provides a REST API to manage items.

## Features
CRUD operations for items 
Connects to a MySQL database
Simple API for frontend integration

## Tech Stack
Java / Spring Boot
MySQL
Maven

## Getting Started

### 1. Clone the repo
git clone https://github.com/shibambocollins/ProdManager

## 2. Configure database
Make sure you have MySQL running and update application.properties with your username and password

 ## 3. Run the application:
mvn spring-boot:run
API will run at http://localhost:8081/api/products

##  4. Notes
This project is for learning purposes.
Use it with the frontend to test the full application: https://github.com/shibambocollins/prod-manager-frontend
